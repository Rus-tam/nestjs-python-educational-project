## Описание

Учебный проект по запуску скриптов python посредством nodejs

## Установка приложения

```bash
$ npm install
```

## Запуск приложения

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
